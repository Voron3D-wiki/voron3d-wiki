# Tri-Zero (T0)

Official Tri-Zero link: [https://github.com/zruncho3d/tri-zero](https://github.com/zruncho3d/tri-zero)

![Tri-Zero render](https://github.com/zruncho3d/tri-zero/raw/main/Renders/alpha-6/iso-red.png)

## What is a Tri-Zero

Designed by Zruncho, the Tri-Zero (often called T0) is a Voron V0 mod that adds automatic bed leveling using a triple-belted Z system. It can be used to upgrade an existing V0 or to build a T0 from a V0 kit with additional parts.

## Quick Facts

- Base platform: Voron V0 / V0.2
- Z motion: Triple belted Z (three independent Z axes)
- Bed leveling: Klicky-style ZeroClick detachable probe
- Build size (stock): 120 x 120 mm
- Build size (Plus50): 170 x 170 mm (adds 50 mm X/Y travel)

## Highlights

- Improved bed stability and Z quality from three belted Z axes
- Tool-less belt attachment and single-screw tensioning on all Z axes
- Built-in Wago mounts for easy bed wiring and removal
- Multiple display options supported (touchscreen, Mini 12864, V0 Mini Display, or blank panel)
- Supports large NEMA14 and mid-to-large NEMA17 Z motors
